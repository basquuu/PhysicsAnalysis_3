- 주제 : 초등학교 생존수영 활성화를 위한 수영장 입지 선정

- 국민 체육 진흥 공단 : 전국공공 체육시설 데이터, 지역별 스포츠 동호회 현황

- 서울시종합방재센터 : 서울시 안전센터 관할 위치정보
